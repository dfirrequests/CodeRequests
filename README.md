# CodeRequests
This is a location where individuals within the DFIR can ask for coding requests. Requests are submitted in the form of issues.

Here is a quick "key"
Issue Numer
<number>

[O] = Open
[W] = In-progress
[X] = Closed

Example: 1[X] = Issue number one and it's closed (i.e., complete)
